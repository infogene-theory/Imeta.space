$$G\frac{m_1 m_2}{r^2} \approx |\nabla S| \cdot m \cdot k_B \cdot \ln\left(\frac{\delta S}{\langle S\rangle}\right)$$

### 6.2 自动温控系统（PID控制器）

```python
class EntropyFluctuationController:
    """基于熵涨落的PID温控系统"""
    
    def __init__(self, target_delta_S_ratio=0.45, Kp=1.0, Ki=0.1, Kd=0.05):
        self.target = target_delta_S_ratio  # 目标熵涨落比（太极态中心）
        self.Kp, self.Ki, self.Kd = Kp, Ki, Kd
        self.integral = 0
        self.last_error = 0
    
    def control(self, current_C, current_delta_S_ratio, dt=1.0):
        """
        输入: 
          - current_C: 当前相干度
          - current_delta_S_ratio: 当前 δS/⟨S⟩
        输出: 
          - 控制量(正=加热↑δS, 负=降温↓δS)
        """
        error = self.target - current_delta_S_ratio
        
        # P: 比例项（当前误差）
        P = self.Kp * error
        
        # I: 积分项（历史累积误差）
        self.integral += error * dt
        I = self.Ki * self.integral
        
        # D: 微分项（误差变化率）
        D = self.Kd * (error - self.last_error) / dt
        
        self.last_error = error
        control_signal = P + I + D
        
        # 根据相干度调整控制强度
        if current_C > 0.85:  # 过度秩序，需要强力加热
            control_signal *= 1.5
        elif current_C < 0.4:  # 过度混乱，需要强力降温
            control_signal *= 1.5
        
        return control_signal
```
