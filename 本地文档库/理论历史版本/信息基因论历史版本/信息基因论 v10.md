## **第三部分：核心测量体系（继承v9.0，新增自旋测量）**

### **3.1 四指标计算协议**

#### **指标1：频率秩序度$O_{freq}$**（继承v9.0）
$$O_{freq} = 1 - \frac{S_{freq}}{\log_2 N}$$

**物理意义**：节律的稳定性（0=白噪声，1=单频）

---

#### **指标2：尺度不变指数$I_{STI}$**（继承v9.0）
$$I_{STI} = 1 - \text{CV}(O_{freq}^{(1)}, O_{freq}^{(2)}, ..., O_{freq}^{(M)})$$

**物理意义**：跨尺度一致性（0=噪声，1=完美自相似）

---

#### **指标3：旋向一致性$\eta$**（新增）
**定义**：
$$\eta = \frac{\left|\sum_i \vec{s}_i\right|}{\sum_i |\vec{s}_i|}$$

其中$\vec{s}_i$为第$i$个单元的自旋向量。

**物理意义**：
- $\eta=0$：自旋方向完全随机抵消
- $\eta=1$：所有自旋完全对齐
- $\eta \in [0.8, 1]$：强自旋锁定（生命态、涡旋态）

**测量方法**：

| 系统类型 | 自旋向量定义 | 测量技术 |
|---------|-------------|---------|
| 量子系统 | 电子自旋$\vec{s}=\frac{\hbar}{2}\vec{\sigma}$ | Stern-Gerlach实验 |
| 分子系统 | 手性角度$\theta_{\text{chiral}}$ | 圆二色光谱(CD) |
| 流体系统 | 涡度向量$\vec{\omega}=\nabla \times \vec{v}$ | PIV流场测量 |
| 神经系统 | 相位向量$\vec{\phi}=e^{i\theta(t)}$ | EEG相位分析 |

**标准误差**：
$$\sigma_\eta \approx \frac{\sqrt{1-\eta^2}}{\sqrt{N}}$$

---

#### **指标4：涌现速率$E_{rate}$**（继承v9.0）
$$E_{rate} = \frac{1}{n-1}\sum_{i=2}^n \frac{|\Delta H_i|}{|H_i|}$$

**物理意义**：新频率峰出现的速率（结构演化速度）

---

### **3.2 综合相干度$C$**（v10.0更新）
**统一公式**：
$$C = \sqrt[3]{O_{freq} \cdot I_{STI} \cdot (1 + \alpha \eta)}$$

**形态判别矩阵**：

            spin_vectors: 自旋向量数组 (N, 3)，可选
        """
        self.signal = np.array(signal)
        self.fs = sampling_rate
        self.scales = scales or [2**i for i in range(1, 9)]
        self.spin_vectors = spin_vectors
        
    def compute_spin_alignment(self, 
                               spin_vectors: Optional[np.ndarray] = None) -> Tuple[float, Dict]:
        """
        计算旋向一致性η（新增）
        
        参数:
            spin_vectors: (N, 3) 自旋向量数组
        
        返回:
            eta: 旋向一致性
            details: 详细信息
        """
        if spin_vectors is None:
            spin_vectors = self.spin_vectors
            
        if spin_vectors is None:
            return 0.0, {'error': 'No spin vectors provided'}
        
        # 归一化自旋向量
        norms = np.linalg.norm(spin_vectors, axis=1, keepdims=True)
        normalized = spin_vectors / (norms + 1e-12)
        
        # 计算净自旋向量
        net_spin = np.sum(normalized, axis=0)
        net_magnitude = np.linalg.norm(net_spin)
        
        # 旋向一致性
        eta = net_magnitude / len(normalized)
        
        # 标准误差
        sigma_eta = np.sqrt(1 - eta**2) / np.sqrt(len(normalized))
        
        # 主旋向
        if net_magnitude > 0:
            main_direction = net_spin / net_magnitude
        else:
            main_direction = np.array([0, 0, 0])
        
        return eta, {
            'net_spin': net_spin,
            'net_magnitude': net_magnitude,
            'main_direction': main_direction,
            'std_error': sigma_eta,
            'n_vectors': len(normalized)
        }
    
    def compute_phase_coherence(self) -> Tuple[float, Dict]:
        """
        基于Hilbert变换的相位相干度（辅助η计算）
        """
        # Hilbert变换提取瞬时相位
        analytic_signal = hilbert(self.signal)
        instantaneous_phase = np.angle(analytic_signal)
        
        # 相位向量
        phase_vectors = np.column_stack([
            np.cos(instantaneous_phase),
            np.sin(instantaneous_phase),
            np.zeros_like(instantaneous_phase)
        ])
        
        # 计算相位一致性（等效于旋向一致性）
        eta_phase, details = self.compute_spin_alignment(phase_vectors)
        
        return eta_phase, details
    
    def classify_coherence_mode(self, 
                                O_freq: float,

        返回:
            mode: 形态类型
            confidence: 置信度
        """
        # 综合相干度
        alpha = 0.5  # 默认耦合系数
        C = (O_freq * I_STI * (1 + alpha * eta)) ** (1/3)
        
        # 形态判别逻辑
        if C > 0.9 and O_freq > 0.90 and I_STI > 0.95 and eta < 0.2:

            confidence = min(C, (O_freq + I_STI) / 2)

            mode = "Life"
            confidence = min(C, (O_freq + I_STI + eta) / 3)
            
        elif C > 0.80 and 0.65 < O_freq < 0.80 and 0.75 < I_STI < 0.88 and eta > 0.90:
            mode = "Vortex"
            confidence = min(C, (I_STI + eta) / 2)
            
        elif C > 0.80 and 0.72 < O_freq < 0.88 and 0.80 < I_STI < 0.90 and 0.60 < eta < 0.85:
            mode = "Intelligence"
            confidence = min(C, (O_freq + I_STI) / 2)
            
        elif 0.6 < C < 0.8:
            mode = "Critical"
            confidence = C
            
        else:
            mode = "Disordered"
            confidence = 1 - C
        
        return mode, confidence

        """

        """
        results = {
            'metadata': {
                'igt_version': '10.0',

                'sampling_rate': self.fs

        }

        if self.spin_vectors is not None:
            eta, eta_details = self.compute_spin_alignment()
            eta_source = 'external_vectors'
        else:

            eta, eta_details = self.compute_phase_coherence()
            eta_source = 'phase_coherence'
        
        results['eta'] = {
