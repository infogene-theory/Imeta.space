**温度稳定机制**：
```
无水行星（火星）:

  ↓
有水行星（地球）:
外部辐射变化 → 水体吸收/释放热量 → 温度变化被极大衰减
  ↓
δT_表面 = δQ / (m·c) 
  = δQ / (m_水·4.2) << δQ / (m_岩·1.0)
```

**IGT表达**：
- 水是天然的"熵涨落衰减器"
- 增强系统热容 C_V → 降低温度波动 δT/T → 提升相干度 C
- 公式：
  \[
  C_V^{\text{水圈}} = m_{\text{水}} \cdot c_p \gg C_V^{\text{岩石}}
  \]

**观测层证据**：

- 深海热液喷口：现代"化学工厂"（T≈350°C，富含H₂S，FeS）

**温度窗口要求**：
```
过冷（<273K）：反应速率太慢，水冻结
过热（>373K）：有机分子分解
最优区间：273-373K（液态水+适度反应速率）
```

**化学熵动力学**：
| 阶段 | 典型分子 | δS/⟨S⟩ | 熵行为 | 能量来源 |
|------|---------|--------|--------|----------|
| 简单无机 | H₂O, CO₂, NH₃ | 0.8 | 高熵，自由扩散 | 热运动 |
| 小分子有机 | 甲烷, 甲醛 | 0.6 | 局部有序 | UV光解 |

**关键跃迁点：从被动到主动**
```

  反应随环境δS波动，无自主性
    ↓

  - 局部可能稍高温（质子梯度驱动）

  - 等效"温度"：化学势差Δμ
  - 熵作用：高熵源（氧化产生热+CO₂）

**ATP：生命的"能量货币"**
```

def diagnose_system(observations):
    """
    输入：观测数据（时间序列）
