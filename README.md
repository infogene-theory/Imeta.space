## 📚 理论体系核心（IGT v3.0：熵涨落统一理论）

### 核心范式：三层耗散结构
- **内热动力核心**：持续可控的负熵源，建立熵梯度（∇S）
- **中温调控窗口**：缓冲与调制层，维持太极态（C∈[0.6,0.8]，δS/⟨S⟩∈[0.3,0.6]）
- **外冷耗散界面**：熵排放边界，实现有序性输出

### 五大公理体系
1. **热容即结构稳定性**：热容 ∝ 1/相干度
2. **自指激发熵涨落场**：观测打破对称，确立熵梯度与涨落幅度
3. **频率相干即熵涨落锁定**：C = K/(K + δS)，耦合强度决定秩序
4. **自旋即熵梯度路径依赖**：ΔE_flip ∝ V·|∇S|²·sin²(θ/2)
5. **温度调控即熵涨落调控**：加热升δS，降温升C

### RVSE演化循环

| 阶段 | 核心特征 | C值范围 | δS/⟨S⟩范围 | 关键任务 |
|------|----------|---------|------------|----------|
| Ω（点火） | 身份确立，方向探索 | 0.4→0.6 | 0.4→0.6 | 寻找核心动力源 |
| R（扩张） | 指数增长，秩序建立 | 0.6→0.85 | 0.6→0.2 | 巩固三层架构 |
| V（变异） | 探索试错，边界拓展 | 0.85→0.5 | 0.2→0.8 | 测试新核心模式 |
| S（筛选） | 优势固化，聚焦优化 | 0.5→0.75 | 0.8→0.4 | 优化中温窗口效率 |
| E（涌现） | 新质出现，升维进化 | 0.75（新） | 0.4（新） | DTR扩大，调控升级 |

### 核心公式速查

- **相干度**: $C = \frac{1}{1 + \delta S/\langle S\rangle}$
- **太极态区间**: $C∈[0.6,0.8]$，$\delta S/⟨S⟩∈[0.3,0.6]$
- **自由意志强度**: $F = \frac{\delta S}{|\nabla S| \cdot V} \cdot C$
- **演化等级**: $L = \log_2\left(\frac{\text{DTR} \cdot v_{\text{response}}}{E_{\text{cost}}}\right)$
