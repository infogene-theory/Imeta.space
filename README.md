## 📚 理论体系核心（IGT v20）

### 五大公理
1. **能量惯性即热容**: 系统越大，改变越难
2. **温度场自指激发**: Ω₀通过观测触发涌现
3. **频率相干即温度锁定**: 秩序源于温度相干
4. **自旋即温度梯度路径**: 路径依赖性的根源
5. **温度调控即演化能力**: 生命=维持ΔT>0的开放系统

### RVSE热力学循环
```
Ω₀（热平衡态）
    ↓ [触发]
Ω（特征温度确立）
    ↓
R（等温扩张）→ V（临界探索）→ S（筛选冷却）
    ↓                              ↓
E（涌现跃迁）                    D（死亡回归）
```

### 三个核心公式
**有效温度**（系统健康度）:
```
T_eff = (T_t × T_s × T_d × T_i)^(1/4)
```

**相干度**（秩序水平）:
```
C = T₀ / (⟨T⟩ + σ_T)
```

**阴阳相图坐标**:
```
x轴: 相干度 C (0-1)
y轴: 温度涨落 δT/T₀ (0-1)
```
